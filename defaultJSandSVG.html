How does font awesome work - JS + SVG v6

  1. Download the icons
  2. Put the js and svg into your project folder
  3. Add a link to fontawesome.js
  4. Add a link to the solid.js or the brands.js
  5. Find the appropriate icon at https://fontawesome.com/v6/icons
  6. Put it in an i or span element 
    <i class="fa-solid fa-bicycle"></i>
